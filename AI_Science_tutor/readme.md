# **AI Science Tutor**

**Overview**  
This implementation demonstrates streaming and non-streaming examples of OpenAIAssistantsClient with agent-framework where we are building an AI Science Tutor which explains complex science topics to students using analogies and examples.

**Sequential orchestration workflow**  
<img src="https://github.com/cloudalchemy-ai/MAF/blob/main/AI_Science_tutor/image.jpg?raw=true" alt="Sequential Orchestration Workflow" width="700"/>

**Key Features**

1. OpenAIAssistantsClient  
2. OpenAI Credentials

**Prerequisites**

Before starting, make sure you have:

* OpenAI Credentials in env.  
  * OPENAI\_CHAT\_MODEL\_ID\="gpt-4o"  
  * OPENAI\_API\_KEY\="Your API Key"


* Agent-framework installed.  
  * pip install agent-framework
